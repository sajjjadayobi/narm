Model¶

  • Nureal Network: Fully Connected
    • Formula $$X_2 \leftarrow ReLU(X * \theta_1 + \beta_1)$$ $$ score \leftarrow X_2* \theta_2 + \beta_2 $$

  • compute probablity of each class with softmax function

Loss & Optimization¶

  • compute loss and update parameters

  • Start with random initial parameters
  • At each iteration:
  • Optimization with GD: $$ \theta \leftarrow \theta_{old} - \alpha * \ \nabla Loss() $$
